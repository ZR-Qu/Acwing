# 算法刷题笔记 | Algorithm Notes

[![Status](https://img.shields.io/badge/Status-In%20Progress-green)](https://github.com/ZR-Qu/Acwing)

本项目用于存放在刷算法题过程中的代码和笔记，主要记录在 Acwing、洛谷 (Luogu) 等平台以及HUST国产化数据平台(https://sino.cberse.cn/)上遇到的各种算法题型。

This repository is used to store my personal code and notes for algorithm problems, primarily from platforms like Acwing and Luogu.

---

